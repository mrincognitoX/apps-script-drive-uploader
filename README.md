# 🚀 آپلودر پیشرفته لینک به گوگل درایو
**Google Drive URL Uploader (Apps Script)**

یک پروژه قدرتمند و حرفه‌ای مبتنی بر **Google Apps Script** که به شما امکان می‌دهد فایل‌ها را از طریق لینک مستقیم (URL) مستقیماً به حساب گوگل درایو خود منتقل کنید. این ابزار با استفاده از رابط کاربری گرافیکی (Web App) و معماری آپلود تکه‌تکه (Resumable Upload)، محدودیت‌های معمول گوگل در انتقال فایل‌های حجیم را دور می‌زند.

---

## ✨ ویژگی‌های کلیدی

* 🌐 **رابط کاربری تحت وب (GUI):** دارای پنل گرافیکی زیبا و ریسپانسیو (طراحی شده با Tailwind CSS).
* 📦 **پشتیبانی از فایل‌های حجیم:** استفاده از Google Drive API v3 و سیستم `Resumable Upload` برای دور زدن محدودیت ۵۰ مگابایتی و آپلود فایل‌های بزرگ (مثلاً ۵۰۰ مگابایتی) به صورت تکه‌های ۲۰ مگابایتی.
* 📁 **مدیریت هوشمند پوشه‌ها:** کافیست نام پوشه مقصد را وارد کنید؛ اگر پوشه وجود نداشته باشد، اسکریپت به صورت خودکار آن را می‌سازد.
* ⚡ **آپلود دوگانه:** تشخیص خودکار حجم فایل؛ استفاده از دانلود یکپارچه برای فایل‌های کوچک و دانلود/آپلود تکه‌ای برای فایل‌های بزرگ.
* 📊 **لاگ‌گیری زنده:** نمایش لحظه‌ای وضعیت آپلود، درصد پیشرفت و خطاهای احتمالی در محیط کاربری.
* 🛡️ **مدیریت خطاهای سرور:** دور زدن خطاهای رایج مثل `Method Not Allowed` با استفاده از متدهای هوشمند `GET` با هدر `Range: bytes=0-1`.

---

## 🛠️ آموزش نصب و راه‌اندازی

برای استفاده از این پروژه، نیازی به سرور یا هاست ندارید؛ تمام کدها روی فضای ابری رایگان گوگل (Google Apps Script) اجرا می‌شوند.

### مرحله ۱: ایجاد پروژه جدید
۱. به [Google Apps Script](https://script.google.com/) بروید و روی **New Project** کلیک کنید.
۲. فایل پیش‌فرض را به `Code.gs` تغییر نام دهید و کدهای بک‌اند را در آن قرار دهید.
۳. از منوی کناری (آیکون **+**)، یک فایل `HTML` ایجاد کرده و نام آن را دقیقاً `Index` بگذارید. کدهای فرانت‌اند (رابط کاربری) را در آن کپی کنید.

### مرحله ۲: تنظیم مجوزها (Manifest)
برای جلوگیری از خطای `401 Unauthorized` در فایل‌های حجیم، باید مجوزهای دسترسی را صراحتاً در پروژه تعریف کنیم:
۱. از منوی سمت چپ، روی آیکون **چرخ‌دنده ⚙️ (Project Settings)** کلیک کنید.
۲. تیک گزینه **Show "appsscript.json" manifest file in editor** را روشن کنید.
۳. به بخش **Editor (کد ⟨ ⟩)** برگردید و روی فایل `appsscript.json` کلیک کنید.
۴. کدهای زیر را به طور کامل جایگزین محتوای آن کرده و ذخیره کنید:

```json
{
  "timeZone": "Asia/Tehran",
  "dependencies": {},
  "webapp": {
    "access": "MYSELF",
    "executeAs": "USER_DEPLOYING"
  },
  "exceptionLogging": "STACKDRIVER",
  "oauthScopes": [
    "[https://www.googleapis.com/auth/drive](https://www.googleapis.com/auth/drive)",
    "[https://www.googleapis.com/auth/script.external_request](https://www.googleapis.com/auth/script.external_request)"
  ]
}
